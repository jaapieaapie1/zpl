# ip.snmpv3.admin.priv_protocol



This command returns the assigned privacy protocol to be used for the SNMPv3 admin user. It will report
either `"AES"` or `"DES"` .


**Getvar**


To retrieve the assigned privacy protocol:

```
       ! U1 getvar "ip.snmpv3.admin.priv_protocol"

```
