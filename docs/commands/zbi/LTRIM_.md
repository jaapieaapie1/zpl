# LTRIM$




ZBI Commands


This function removes leading spaces from a string.


**Format**
```
   LTRIM$(A$)
```

**Parameters**
`(A$) =` the string to convert.

**Returns**
The string in `A$` with no spaces.

**Example**


This is an example of how to use the LTRIM$(A$)command:

```
   10 LET A$=" Hello"
   20 PRINT LTRIM$(A$)
   RUN
   Hello

```
