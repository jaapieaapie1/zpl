# ip.https.port



Sets the port that will listen for HTTPS connections.


**Setvar**


To set the command:

```
       ! U1 setvar "ip.https.port" "value"

```

**Values**


Any valid https port from 0-65535.


**Default**

```
          "443"

```

**Getvar**


To have the printer return the current setting value:

```
       ! U1 getvar "ip.https.port"

```

1263
