# mcr.crypt.algorithm



Retrieves the MCR encryption algorithm. The return value applies only if `"mcr.crypt.enabled"` is
`"on"` .


**Getvar**


To retrieve the MCR encryption algorithm:

```
       ! U1 getvar "mcr.crypt.algorithm"

```

**Values**

              - `""`

              - `"3des"`

              - `"aes"`

**Default**
NA
