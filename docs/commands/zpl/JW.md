# ^JW




ZPL Commands


`^JW` sets the ribbon tension for the printer it is sent to.


**Set Ribbon Tension**

**Format:** `^JWt`

|Parameters|Details|
|---|---|
|`t =` tension|**Values:**<br>`L =` low<br>`M =` medium<br>`H =` high<br>**Default:** a value must be specified|



**Comments:** `^JW` is used only for **PAX** series printers.
