# ip.dhcp.dhcpv6_duid



This command retrieves the DHCPv6 DUID that the printer uses in DHCPv6 communication. It is listed in
groups of quad hex digits separated by colons. The value is used for both wired and wireless DHCPv6.


**Getvar**


To retrieve the DHCPv6 DUID value:

```
       ! U1 getvar "ip.dhcp.dhcpv6_duid"

```

1229
