# ^WA




ZPL Network Commands


Use this command to set the values for the receive and transmit antenna.


**Set Antenna Parameters**


**Format:** ^WAa,b

|Parameters|Details|
|---|---|
|`a` = receive antenna|**Values:**<br>`D =`diversity<br>`L =`left<br>`R =`right<br>**Default:**`D`|
|`b` = transmit antenna|**Values:**<br>`D =`diversity<br>`L =`left<br>`R =`right<br>**Default:**`D`|
