# ip.mirror.version



This command retrieves the mirror code build date.


**Getvar**


To retrieve the mirror code build date:

```
       ! U1 getvar "ip.mirror.version"

```

**Example**

In this example, the `getvar` retrieves the mirror code build date.

```
       ! U1 getvar "ip.mirror.version"

```

1289
