# MAX




ZBI Commands


This function returns the greater value between two numbers.


**Format**
```
   MAX(X,Y)
```

**Parameters**
`X` = the first number to compare

`Y` = the second number to compare

**Returns**
The greater of `X` or `Y.`

**Example**
This is an example of how to use the MAX(X,Y)command:

```
   10 LET A=-2
   20 LET B=1
   30 PRINT MAX(A,B)
   RUN
   1

```
