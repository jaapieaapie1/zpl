# ISERROR




ZBI Commands


This function returns a non-zero value if there is an internal error set in the printer. Otherwise, the numeral
returned will 0.


**Format**
```
   ISERROR
```

**Parameters**
N/A

**Returns**
0 for no errors; 1 if there is an error.

**Example**
Here is an example of the `ISERROR` command.

```
   10 PRINT ISERROR
   RUN
   0

```
