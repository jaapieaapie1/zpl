# device.ltu_installed



This command checks to see if a Liner Take-Up unit is installed.


**Getvar**


To check if the Liner Take-Up unit is installed or not:

```
       ! U1 getvar "device.ltu_installed"

```

**Values**

              - `"Yes"`

              - `"No"`

**Default**
```
          "No"

```
