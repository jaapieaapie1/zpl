# wlan.ip.dns.servers_user_value



This command displays the value that the user has set for wlan.ip.dns.servers with static (IPv6)/permanent
(IPv4) acquisition mode with a WLAN connection.


**Getvar**


To retrieve the value:

```
       ! U1 getvar "wlan.ip.dns.servers_user_value"

```
