# ip.snmpv3.monitor.auth_protocol



This command returns the assigned authentication protocol to be used for the SNMPv3 monitor user. It will
report either `"MD5"` or `"SHA"` .


**Getvar**


To retrieve the assigned authentication protocol:

```
       ! U1 getvar "ip.snmpv3.monitor.auth_protocol"

```
