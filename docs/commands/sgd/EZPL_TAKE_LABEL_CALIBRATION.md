# ezpl.take_label_calibration



This command initiates the calibration sequence of the take-label sensor. The printer returns the following
statuses during the calibration process: `starting`, `inProgress`, and `complete` .


**Setvar**


To initiate the take-label sensor calibration sequence:

```
       ! U1 setvar "ezpl.take_label_calibration"

```

**Values**


NA


**Default**
NA
