# ^NT




ZPL Network Commands


Use this command to set the Simple Mail Transfer Protocol (SMTP) parameters. This allows you to set the email settings for alerts.


**Set SMTP**


**Format:** ^NTa,b

|Parameters|Details|
|---|---|
|`a` = SMTP server address|**Values:**Any properly formatted server address in the`xxx.xxx.xxx.xxx`<br>format|
|`b` = print server domain|**Values:**Any properly formatted print server domain name. A domain name<br>is one or more labels separated by a period (“dot”), and a label consists of<br>letters, numbers, and hyphens. An example of a domain name is zebra.com|
