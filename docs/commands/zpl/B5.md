# ^B5




ZPL Commands


The `^B5` command is supported in all printers as a resident barcode. Accepted barcode characters are 0-9.


**Planet Code Barcode**

**Format:** `^B5o,h,f,g`









|Parameters|Details|
|---|---|
|`o =` orientation code|**Values:**<br>`N =` normal<br>`R =` rotated<br>`I =` inverted 180 degrees<br>`B =` read from bottom up, 270 degrees<br>**Default:** current`^FW` value|
|`h =` barcode height (in<br>dots)|**Values:** `1` to`9999`<br>**Default:** value set by`^BY`|
|`f =` interpretation line|**Values:**<br>`N =` no<br>`Y =` yes<br>**Default:** `N`|
|`g =` determines if<br>the interpretation line<br>is printed above the<br>barcode|**Values:**<br>`N =` no<br>`Y =` yes<br>**Default:** `N`|
