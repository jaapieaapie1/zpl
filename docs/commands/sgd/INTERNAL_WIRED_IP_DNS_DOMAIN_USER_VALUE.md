# internal_wired.ip.dns.domain_user_value



This command displays the value that the user has set to `internal_wired.ip.dns.domain` . Whether
this value is used will depend on the rules for `internal_wired.ip.dns.domain` .


**Getvar**


To retrieve the domain name(s) from an internal wired print server:

```
       ! U1 getvar "internal_wired.ip.dns.domain_user_value"

```
