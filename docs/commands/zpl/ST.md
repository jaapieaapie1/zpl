# ^ST




ZPL Commands


The `^ST` command sets the date and time of the Real-Time Clock.


**Set Date and Time (for Real-Time Clock)**

**Format:** `^STa,b,c,d,e,f,g`

|Parameters|Details|
|---|---|
|`a =` month|**Values:** `01` to`12`<br>**Default:** current month|
|`b =` day|**Values:** `01` to`31`<br>**Default:** current day|
|`c =` year|**Values:** `1998` to`2097`<br>**Default:** current year|
|`d =` hour|**Values:** `00` to`23`<br>**Default:** current hour|
|`e =` minute|**Values:** `00` to`59`<br>**Default:** current minute|
|`f =` second|**Values:** `00` to`59`<br>**Default:** current second|
|`g =` format|**Values:**<br>`A =` a.m.<br>`P =` p.m.<br>`M =` 24-hour military<br>**Default:** `M`|



For more details on set date and time, see Real Time Clock on page 1595.
