# ip.mirror.fetch



This command forces a mirroring update sequence.


**Setvar**


To force a mirroring update sequence:

```
       ! U1 setvar "ip.mirror.fetch" ""

```

**Do**


To force a mirroring update sequence:

```
       ! U1 do "ip.mirror.fetch" ""

```

This `setvar` example forces a mirroring update sequence.
