# ip.snmpv3.monitor.name



This command returns the assigned username for the SNMPv3 monitor user (a user with read-only access).
The value is a string with up to 32 characters. By default, the value is an empty string.


**Getvar**


To retrieve the assigned username:

```
       ! U1 getvar "ip.snmpv3.monitor.name"

```
