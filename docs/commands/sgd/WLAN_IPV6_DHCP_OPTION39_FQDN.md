# wlan.ipv6.dhcp.option39_fqdn



This command retrieves the fully qualified domain name (FQDN) of the printer as given by the DHCPv6
server for option 39 with a WLAN connection. The value is a string up to 127 characters.


**Getvar**


To retrieve the FQDN:

```
       ! U1 getvar "wlan.ipv6.dhcp.option39_fqdn"

```
