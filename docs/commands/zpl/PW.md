# ^PW




ZPL Commands


The `^PW` command allows you to set the print width.


**Print Width**
```
^PWa

|Parameters|Details|
|---|---|
|`a =` label width (in dots)|**Values:** `2`, to the width of the label<br>If the value exceeds the width of the label, the width is set to the label’s<br>maximum size.<br>**Default:** last permanently saved value|


```

**Comments**


This command is ignored on the HC100™ printer.
