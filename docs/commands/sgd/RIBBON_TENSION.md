# ribbon.tension



This command sets the tension of the ribbon.


**Getvar**


To report the current tension of the ribbon:

```
       ! U1 getvar “ribbon.tension”

```

**Values**

              - `"low"`

              - `"medium"`

              - `"high"`

**Default**
```
          "high"

```

**Setvar**


To specify the tension of the ribbon:

```
       ! U1 setvar “ribbon.tension" "value"

```

**Values**

              - `"low"`

              - `"medium"`

              - `"high"`
