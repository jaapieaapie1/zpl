# head.resolution.in_dpi



Returns the resolution of the print head in dots per inch as an integer.


**Getvar**


To return the resolution of the print head in dots per inch:

```
       ! U1 getvar "head.resolution.in_dpi"

```

**Values**

              - `"200"`

              - `"300"`

              - `"600"`
