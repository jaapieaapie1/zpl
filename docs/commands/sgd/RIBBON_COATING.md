# ribbon.coating



This command sets the type of ribbon coating used.


**Getvar**


To determine the setting for the ribbon coating:

```
       ! U1 getvar “ribbon.coating”

```

**Values**

              - `"ink side in"`

              - `"ink side out"`


**Setvar**


To configure the printer for the printer ribbon:

```
       ! U1 setvar “ribbon.coating" "value"

```

**Values**

              - `"ink side in"`

              - `"ink side out"`
