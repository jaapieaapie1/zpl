# ip.snmpv3.monitor.priv_protocol



This command returns the value of the privacy protocol to be used for the SNMPv3 monitor user. It will
report a value of either `"AES"` or `"DES"` .


**Getvar**


To retrieve the privacy protocol value:

```
       ! U1 getvar "ip.snmpv3.monitor.priv_protocol"

```
