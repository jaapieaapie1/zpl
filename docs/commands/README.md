# ZPL/ZBI/SGD Commands Documentation

This directory contains extracted command documentation from the ZPL II Programming Guide.

## Organization

- [ZPL Commands](zpl/README.md) - 199 commands
- [SGD Commands](sgd/README.md) - 867 commands
- [ZBI Commands](zbi/README.md) - 104 commands
- [Other Documentation](other/) - 30 files

## Command Formats

- **ZPL Commands**: Start with `^` (caret) or `~` (tilde)
- **SGD Commands**: Use dot notation (e.g., `device.friendly_name`)
- **ZBI Commands**: BASIC-like commands for Zebra Basic Interpreter
