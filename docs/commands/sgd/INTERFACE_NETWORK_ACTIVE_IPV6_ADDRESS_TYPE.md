# interface.network.active.ipv6.address_type



This command displays the method, either static or automatic, to get IPv6 addresses with wired or wireless
networking.


**Getvar**


To retrieve the active network address type:

```
       ! U1 getvar "interface.network.active.ipv6.address_type"

```

**Example**

```
       ! U1 getvar "interface.network.active.ipv6.address_type"

```

**Result**

```
          "auto"

```
