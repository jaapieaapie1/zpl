# MIN




ZBI Commands


This function returns the smaller value of two numbers.


**Format**
```
   MIN(X,Y)
```

**Parameters**
`X` = the first number to compare

`Y` = the second number to compare

**Returns**
The smaller of `X` or `Y.`

**Example**


This is an example of how to use the MIN(X,Y)command:

```
   10 LET A=-2
   20 LET B=0
   30 PRINT MIN(A,B)
   RUN
   -2

```
