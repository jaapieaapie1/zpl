# wlan.ipv6.dhcp.lease.length



This command retrieves the value of the lease duration for an IP address provided by the DHCPv6 in
seconds with a WLAN connection. The value is a whole number up to 4,294,967,295 in seconds.


**Getvar**


To retrieve the lease length:

```
       ! U1 getvar "wlan.ipv6.dhcp.lease.length"

```

1440
