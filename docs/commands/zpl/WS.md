# ^WS




ZPL Network Commands


Use this command to set the wireless radio card values for ESSID, Operating Mode, and Card Preamble.


**Set Wireless Radio Card Values**

**Format:** `^WSe,o,p,h,i,j,k`





|Parameters|Details|
|---|---|
|`e` = ESSID value|**Values:**Any value up to 32 characters, including all ASCII and Extended<br>ASCII characters, including the space character. When this parameter is left<br>blank, the ESSID is not changed.<br>**Default:**125|
|`o` = operating mode|**Values:**I (Infrastructure), A (Adhoc)<br>**Default:**I|
|`p` = wireless radio card<br>preamble|**Values:**<br>`L =`long<br>`S =`short<br>**Default:**L|
|h = wireless pulse<br>This parameter is<br>supported in firmware<br>version V60.15.x,<br>V50.15.x, R6x.15.x,<br>R53.15.x, ZSPx, or later.|Adds a pulse to the network traffic generated by the printer. This pulse<br>is necessary with some network configurations to keep the printer<br>online.**Values:**<br>`0 =`disabled<br>`1 =`enabled<br>**Default:**1|
|i = wireless pulse interval<br>This parameter is<br>supported in firmware<br>version V60.15.x,<br>V50.15.x, R6x.15.x,<br>R53.15.x, ZSPx, or later.|Sets the interval at which the wireless pulse is sent when the wireless pulse<br>feature is enabled.<br>**Values:**5 to 300 seconds<br>**Default:**15|
|j = channel mask<br>This parameter is<br>supported in firmware<br>version X60.15.x,<br>V50.15.x, or later.|For commonly used channel masks, see**Table 13 on page 424**.<br>**Values:**4 Hexadecimal digits preceded by “0x” (0x0000 to 0xFFFF)<br>**Default:**0x7FF|
