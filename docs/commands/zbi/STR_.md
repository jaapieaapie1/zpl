# STR$




ZBI Commands


This function converts a number to a string.


**Format**
```
   STR$(X)
```

**Parameters**
`X` = the number to convert to a string

**Returns**
A string representing `X` .

**Example**


This is an example of how to use the STR$(X)command:

```
   10 LET A=53
   20 PRINT STR$(A)
   RUN
   53

```
