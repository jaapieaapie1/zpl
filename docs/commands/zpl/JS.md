# ^JS




ZPL Commands


The `^JS` command specifies which sensor to use.


**Sensor Select**

**Format:** `^JSa`


**NOTE:** This command is only for use with the S4M and some Z Series printers. This command is
ignored on Zebra ZM400/ZM600 and RZ400/RZ600 printers.
