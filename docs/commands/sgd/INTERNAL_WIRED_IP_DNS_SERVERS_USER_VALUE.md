# internal_wired.ip.dns.servers_user_value



This command displays the value that the user has set to `internal_wired.ip.dns.servers` . Whether
this value is used will depend on the rules for `internal_wired.ip.dns.servers` .


**Getvar**


To retrieve the domain name server(s):

```
       ! U1 getvar "internal_wired.ip.dns.servers_user_value"

```
