# interface.network.active.ipv6.dhcp_server_duid



This command returns the DHCP unique identifier (DUID), as provided by the DHCPv6 server that provided
the printer network information.


**Getvar**


To retrieve the DUID:

```
       ! U1 getvar "interface.network.active.ipv6.dhcp_server_duid"

```
