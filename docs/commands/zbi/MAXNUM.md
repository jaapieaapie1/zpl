# MAXNUM



This function returns the largest number permitted by this machine: 2,147,483,647.


**Format**
```
          MAXNUM
```

**Parameters**
N/A

**Returns**
The largest number that the NUMERIC type can handle (2,147,483,647).

**Example**
This is an example of how to use the MAXNUM command:

```
          10 PRINT MAXNUM
          RUN
          2147483647

```
