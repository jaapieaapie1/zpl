# wlan.8021x.ttls_anonymous_identity



This command returns the anonymous identity string for EAP-TTLS that is used as the unencrypted identity
for a WLAN. The value of the identity string can be up to 32 characters.


**Getvar**


To return the anonymous identity string:

```
       ! U1 getvar "wlan.8021x.ttls_anonymous_identity"

```

**Default**
NA
