# wlan.ipv6.dhcp.lease.last_attempt



This command retrieves the value of the Unix timestamp of the last attempt to obtain a DHCPv6 lease with
a WLAN connection. The value is a whole number up to 4,294,967,295 in seconds.


**Getvar**


To retrieve the time of the last attempt:

```
       ! U1 getvar "wlan.ipv6.dhcp.lease.last_attempt"

```

1439
