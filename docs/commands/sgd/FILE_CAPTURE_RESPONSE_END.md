# file.capture_response.end



This command disables file capture mode.


**Do**

Use this command in conjunction with `file.capture_response.begin` and
`file.capture_response.destination` . The begin and end commands must be issued from the
same port.


To disable file capture mode:

```
       ! U1 do "file.capture_response.end" ""

```

**Values**
NA

**Default**
NA
