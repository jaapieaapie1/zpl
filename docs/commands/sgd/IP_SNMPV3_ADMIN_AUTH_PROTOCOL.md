# ip.snmpv3.admin.auth_protocol



This command returns the assigned authentication protocol to be used for the SNMPv3 admin user. The
value with be either `"MD5"` or `"SHA"` .


**Getvar**


To retrieve the assigned authentication protocol:

```
       ! U1 getvar "ip.snmpv3.admin.auth_protocol"

```
