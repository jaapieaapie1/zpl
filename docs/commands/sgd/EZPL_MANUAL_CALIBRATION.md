# ezpl.manual_calibration



This command initiates a manual calibration sequence.


**Setvar**


To instruct the printer to initiate a manual calibration:

```
       ! U1 setvar "ezpl.manual_calibration" ""

```

**Values**
NA

**Default**
NA


**Do**


To instruct the printer to initiate a manual calibration:

```
       ! U1 do "ezpl.manual_calibration" ""

```

**Values**
NA

**Default**
NA
